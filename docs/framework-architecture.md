# QA Ruby Selenium Framework - Arquitectura

## Tecnologías

- Ruby
- Selenium WebDriver
- Minitest
- Git
- GitHub

## Patrón principal

El framework utiliza Page Object Model (POM), separando la lógica de interacción con las páginas de la lógica de los tests.
